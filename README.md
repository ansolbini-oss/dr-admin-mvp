# DR Admin MVP (KPX Integration)

## Overview
This project is a prototype system designed to validate communication with KPX using OpenADR.

## Goal
To implement a minimal system that supports:
- Event reception
- Response handling
- Power usage data transmission

## Features
- Connection management with KPX
- DR event receive & response
- Power usage data transmission
- Communication logging

## Structure
See details:
- docs/structure.md
- docs/features.md

## Architecture
- Feature-based modular design
- OpenADR communication adapter
- Mock data for testing

## Purpose
This MVP focuses on validating KPX integration before building a full DR operation system.
